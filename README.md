### Game Rules:
- Each player takes turns guessing the movie
- The image starts pixelated and gradually becomes clearer
- Use hints wisely (you get 3 per movie)
- Score points based on:
  - Difficulty level
  - Time taken to guess
  - Number of hints used
  - Current streak
